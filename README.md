# cybertron-2099

Projeto onde um robo é montado e peças nas engrenagens desse robo precisam ser adicionadas, as estatísticas dessas mudanças são atualizadas dinamicamente.

Projeto com foco em manipulação do DOM, entre outros recursos da linguagem JavaScript.

## :heavy_check_mark: Técnologias utilizadas:

- `HTML`
- `CSS`
- `JavaScript`
